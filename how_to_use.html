How to Use the Library Book Issue Tracker

1

Using the HTML/JavaScript Version

The HTML/JavaScript version runs directly in your web browser and stores data in your browser's local storage.

To use it:

  1. Double-click on the library_tracker.html file in your file explorer
  2. Your browser will open and display the Library Tracker application
  3. You can add books, issue them, return them, and view all books using the tabbed interface
  4. Data is automatically saved in your browser's local storage

Note: Data is stored only in your browser. If you clear your browser data or use a different browser, the data will not be available.

2

Using the C++ Version

The C++ version requires a C++ compiler to build and run.

If you have a C++ compiler installed (like g++), you can compile and run it with:

g++ library_tracker_simple.cpp -o library_tracker
./library_tracker

If you're on Windows with MinGW:

g++ library_tracker_simple.cpp -o library_tracker.exe
library_tracker.exe

If you don't have a C++ compiler installed, you can:

  1. Install MinGW for Windows or g++ for Linux/Mac
  2. Use an online C++ compiler like repl.it
  3. Use an IDE like Code::Blocks or Visual Studio that includes a compiler
3

Features

  • Add Books: Add new books with ID, title, and author
  • Issue Books: Mark books as issued to a person with a date
  • Return Books: Return issued books to the library
  • Display Books: View all books and their status
  • Data Persistence: Books are saved between sessions
